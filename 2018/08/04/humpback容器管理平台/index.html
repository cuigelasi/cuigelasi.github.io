humpback容器管理平台

发表于 2018-08-04

前提条件

环境：

docker1.12；禁用selinux
humpback-1：172.16.13.13 
humpback-2：172.16.13.14
humpback-3：172.16.13.15

humpback-1

sed -i "s/^SELINUX=.*/SELINUX=disabled/" /etc/selinux/config 
hostnamectl set-hostname humpback-1 --static
hostnamectl set-hostname humpback-1 --transient
hostnamectl set-hostname humpback-1 --pretty
reboot
yum -y install docker 
systemctl start docker
systemctl enable docker
systemctl stop firewalld

humpback-2

sed -i "s/^SELINUX=.*/SELINUX=disabled/" /etc/selinux/config

reboot

Deploy a private registry

humpback-1

mkdir -p /var/lib/registry
docker run -d --network host --restart=always -v /var/lib/registry/:/var/lib/registry/ --name registry -h registry registry
docker exec -it registry sh
vi /etc/docker/registry/config.yml


curl http://<humpback-1_ip>:5000/v2/_catalog
vim /etc/sysconfig/docker


systemctl restart docker

Deploy The Humpback Site

humpback-1

mkdir -p /opt/app/humpback-web
docker run -d --net=host --restart=always -e HUMPBACK_LISTEN_PORT=8012 -v /opt/app/humpback-web/dbFiles:/humpback-web/dbFiles --name humpback-web -h humpback-web humpbacks/humpback-web:1.3.0
username:admin   passwd:123456

Deploy the Humpback Agent

humpback-1

docker run -d -ti --net=host --restart=always --name=humpback-agent -h humpback-agent -e DOCKER_API_VERSION=v1.21 -e DOCKER_AGENT_IPADDR=0.0.0.0 -e DOCKER_CLUSTER_ENABLED=true -e DOCKER_CLUSTER_URIS=zk://<humpback-1_ip>:2181,<humpback-2_ip>:2181,<humpback-3_ip>:2181 -e DOCKER_CLUSTER_NAME=humpback/center -v /var/run/:/var/run/:rw humpbacks/humpback-agent:1.3.0

Start in single mode

Deploy Humpback Center
