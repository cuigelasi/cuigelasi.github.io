发表于 2018-10-18 | 分类于 K8S

前言

Dashboard 俗称图形界面，仪表盘。是kubernetes的前端页面展示。
Dashboard可以做什么？你在命令行能做的事情，他都能做到，因为他用的就是底层命令。
你在命令行不好实现的，它也能做，因为是图形界面可以更好的进行资源管控。
话不多说，先安装上！

安装K8S Dashboard

如果你还没安装K8S集群，你可以参考我之前的博文去安装。

vim k8s_dashboard.sh

安装步骤很简单按照提示信息一步步执行即可

使用Dashboard

网页进入Dashboard之后，我们可以畅快的使用Dashboard了。

第一步: 登陆kubernetes dashboard

第二步： 点击创建，编写deployment资源

第三步： 上传资源，进行资源创建

第四步： 资源创建完毕，查看资源状态

第五步：进入资源组
